# CareerOS Build Week Demo

```text
cd apps/buildweek-demo
npm install
node server.mjs
```

Open `http://127.0.0.1:4173`.

Run verification with:

```text
npm test
```

This local Build Week demo uses verified public-safe evidence and clearly labeled synthetic demonstration data. It has no external network, LLM/API, authentication, database, or private-source dependency.
